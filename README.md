# Robotic Arm using Arduino

This project demonstrates control of a robotic arm using servo motors and Arduino.

## Components
- Arduino Uno
- Servo motors (4)
- Motor driver module
- External power supply

## Features
- Base rotation
- Shoulder movement
- Elbow movement
- Gripper control

## Working
Each joint of the robotic arm is controlled using servo motor angles programmed in Arduino.

1. Upload the .ino file to Arduino IDE
2. Connect servos to respective pins
3. Power the system
